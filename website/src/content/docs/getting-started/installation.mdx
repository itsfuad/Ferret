## Install (recommended)

<Tabs>
  <TabItem label="macOS / Linux (shell)">
    ```bash
    curl -fsSL https://ferret.brainbird.org/install.sh | sh
    ```
  </TabItem>
  <TabItem label="Windows (PowerShell)">
    ```sh
    irm -useb https://ferret.brainbird.org/install.ps1 | iex
    ```
  </TabItem>
  <TabItem label="Windows (Command Prompt)">
    ```bat
    powershell -NoProfile -ExecutionPolicy Bypass -Command "irm -useb https://ferret.brainbird.org/install.ps1 | iex"
    ```
  </TabItem>
  <TabItem label="Termux">
    ```bash
    curl -fsSL https://ferret.brainbird.org/install-termux.sh | sh
    ```
  </TabItem>
</Tabs>

:::note
The installer downloads a prebuilt release for your OS/arch into `~/.ferret` (Linux/macOS) or `%USERPROFILE%\.ferret` (Windows), and adds the `bin/` directory to your PATH. On Termux, the script builds from source and installs into `$PREFIX`.
You can change the location by setting `FERRET_INSTALL_DIR`. Set `FERRET_ADD_TO_PATH=0` to skip PATH changes.
:::

## Verify installation

```bash
ferret --version
```

## Build from source

If you want to build Ferret yourself, you need Go and a C compiler available on the machine.

```bash
git clone https://github.com/itsfuad/Ferret.git
cd Ferret
./build.sh
```

Windows:

```bat
scripts\build.bat
```

## Running Ferret programs

```bash
ferret yourfile.fer
```
